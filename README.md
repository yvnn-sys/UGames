🎮 UGames
Simple Game Showcase Website

UGames adalah website sederhana yang menampilkan berbagai informasi tentang game dengan tampilan yang clean, ringan, dan mudah digunakan. Project ini dibuat sebagai latihan sekaligus portfolio di bidang front-end development.

🔗 Live Demo:
👉 https://yvnn-sys.github.io/UGames/#

🚀 Fitur Utama
  🎯 Tampilan sederhana dan clean
  🕹️ Menampilkan daftar game
  📱 Responsive (bisa dibuka di berbagai ukuran layar)
  ⚡ Loading ringan dan cepat
  🧭 Navigasi mudah dipahami
  
🛠️ Tech Stack
  HTML5 → Struktur website
  CSS3 → Styling dan layout
  JavaScript → Interaksi dan fitur

📂 Struktur Project
  UGames/
  │── index.html
  │── style.css
  │── script.js
  │── assets/
  │   ├── images/
  │   └── lainnya

⚙️ Cara Menjalankan
git clone https://github.com/yvnn-sys/UGames.git
cd UGames

Lalu buka file index.html di browser.

🎯 Tujuan Project

Project ini dibuat untuk:

Mengasah skill front-end (HTML, CSS, JS)
Belajar membuat UI website yang rapi
Menjadi bagian dari portfolio pribadi
🔥 Pengembangan Selanjutnya

Beberapa ide pengembangan:
